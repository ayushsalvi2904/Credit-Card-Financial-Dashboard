# 💳 Credit Card Financial Dashboard

This project is an interactive Power BI dashboard built to analyze credit card customer behavior, transaction trends, revenue generation, and customer demographics.  
The dashboard helps in understanding customer spending patterns and provides useful business insights using MySQL and Power BI.

---

# 📌 Project Objective

The main objective of this project is to create a weekly credit card dashboard that can track important business metrics and help stakeholders monitor overall credit card performance in a simple and visual way.

---

# 🛠️ Tools & Technologies Used

- Power BI
- MySQL
- SQL
- Power Query
- DAX
- CSV Files

---

# 📂 Dataset Used

The project is built using two datasets:

- **cust_detail** → Customer demographic information
- **cc_detail** → Credit card transaction and financial information

---

# 📊 Dashboard Highlights

- Revenue and transaction analysis
- Weekly performance tracking
- Interest earned analysis
- Customer segmentation
- Income group analysis
- State-wise revenue contribution
- Card category performance
- Education and marital status insights
- Dynamic filters and KPI cards

---

# 📈 Key Insights (Week 53 - 31st Dec)

## Week-over-Week Changes

- Revenue increased by **28.8%**
- Total transaction amount increased by **77%**
- Total transaction count increased by **2.2%**
- Customer count also showed positive growth

---

# 📅 Overall Year-to-Date Performance

- Overall Revenue: **57M**
- Total Interest Earned: **8M**
- Total Transaction Amount: **46M**
- Male customers contributed **31M** in revenue
- Female customers contributed **26M** in revenue
- Blue & Silver credit cards contributed around **93%** of transactions
- TX, NY, and CA contributed around **68%** of total revenue
- Overall activation rate: **57.5%**
- Overall delinquent rate: **6.06%**

# 🧠 Business Insights

- Blue card users contribute the highest revenue among all card categories.
- Customers from high-income groups generate higher transaction values.
- Married customers contribute more revenue compared to single customers.
- Revenue trends vary significantly across different weeks.
- Both male and female customers show strong contribution toward overall revenue.

# 🚀 How to Run the Project

1. Import CSV files into MySQL
2. Run the SQL script
3. Connect MySQL database with Power BI
4. Open the `.pbix` file in Power BI Desktop
5. Refresh the data and explore the dashboard
